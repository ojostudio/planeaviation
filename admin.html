Configure o Firebase para ativar o painel

Este painel usa Firebase (Auth + Firestore + Storage). Para ligá-lo:

  1. Crie um projeto em console.firebase.google.com.
  2. Ative Authentication → E-mail/senha, crie um Firestore e o Storage.
  3. Cole as chaves do app em firebase-config.js.
  4. Publique as regras de firestore.rules e storage.rules.
  5. Crie o primeiro admin (passo a passo no SETUP.md).

Enquanto não configurar, o site público continua funcionando com os dados de exemplo.
